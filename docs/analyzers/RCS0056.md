# RCS0056: A line is too long

| Property | Value      |
| -------- | ---------- |
| Id       | RCS0056    |
| Category | Formatting |
| Severity | None       |

## Summary

This analyzer reports any line that is longer than maximal length.
    
Default maximal length is 140.

## Options

### Max line length

```editorconfig
roslynator_max_line_length = <NUM>
```
