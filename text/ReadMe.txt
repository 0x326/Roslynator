## Roslynator
* A collection of 110+ analyzers and 160+ refactorings for C#, powered by Roslyn.
* [Analyzers](http://github.com/JosefPihrt/Roslynator/blob/master/source/Analyzers/README.md)
* [Refactorings](http://github.com/JosefPihrt/Roslynator/blob/master/source/Refactorings/README.md)
* [Release Notes](http://github.com/JosefPihrt/Roslynator/blob/master/ChangeLog.md)

### Options

* Refactorings can be enabled/disabled in Visual Studio options

![Refactorings Options](/images/RefactoringsOptions.png)

### Distribution

##### Roslynator
* [Extension for Visual Studio 2015](http://visualstudiogallery.msdn.microsoft.com/e83c5e41-92c5-42a3-80cc-e0720c621b5e) that contains both analyzers and refactorings.

##### Roslynator 2017
* [Extension for Visual Studio 2017](http://marketplace.visualstudio.com/items?itemName=josefpihrt.Roslynator2017) that contains both analyzers and refactorings.

##### Roslynator Refactorings
* [Extension for Visual Studio 2015](http://visualstudiogallery.msdn.microsoft.com/a9a2b4bc-70da-437d-9ab7-b6b8e7d76cd9) that contains only refactorings.

##### Roslynator Refactorings 2017
* [Extension for Visual Studio 2017](http://marketplace.visualstudio.com/items?itemName=josefpihrt.RoslynatorRefactorings2017) that contains only refactorings.

##### C# Analyzers
* [NuGet package](http://www.nuget.org/packages/CSharpAnalyzers/) that contains only analyzers.
